Selenium Automation Sprint 2026
This repository contains a collection of Selenium WebDriver automation scripts developed as part of my transition toward a Senior SDET role.

🛠 Tech Stack
Language: Java 17

Tool: Selenium WebDriver (4.25.0)

Build Tool: Maven

IDE: VS Code

🚀 Key Modules Covered
Dropdown Handling: Implementation of the Select class using visible text and values (including custom debugging of DOM typos).

Element Verification: Using isSelected() for state validation of checkboxes and radio buttons.

Synchronization: Implementation of WebDriverWait and ExpectedConditions to handle dynamic loading.
